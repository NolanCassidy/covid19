# COVID

Single page Coronavirus tracking application built using vanilla javascript.

## Getting Started

1. download/clone the github repo

2. install
```
npm install
```

3. start application
```
npm start
```

4. Project can now be viewed at http://localhost:8080/

## Running the tests

Testing is run using Jest.

```
npm test
```
